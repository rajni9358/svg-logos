Python में Nested if स्टेटमेंट

Python nested if statements को सपोर्ट करता है, इसका मतलब है कि हम एक if statement के अंदर एक और if या if...else स्टेटमेंट का उपयोग कर सकते हैं।

ऐसी स्थिति हो सकती है जब आप किसी प्रारंभिक शर्त के true होने के बाद अतिरिक्त शर्तों की जांच करना चाहें। ऐसी स्थिति में आप nested if संरचना का उपयोग कर सकते हैं।

इसके अलावा, एक nested if संरचना के अंदर आप एक if...elif...else संरचना को भी शामिल कर सकते हैं।

Nested if स्टेटमेंट की सिंटैक्स

nested if construct with else condition की सिंटैक्स इस प्रकार होगी −

if boolean_expression1:
    statement(s)
    if boolean_expression2:
        statement(s)

Nested if स्टेटमेंट का फ्लोचार्ट

Python के nested if स्टेटमेंट का फ्लोचार्ट इस प्रकार है −

Nested if स्टेटमेंट का उदाहरण

नीचे दिया गया उदाहरण nested if स्टेटमेंट का कार्य दिखाता है −

num = 36
print("num =", num)
if num % 2 == 0:
    if num % 3 == 0:
        print("Divisible by 3 and 2")

print("....execution ends....")

उपरोक्त कोड चलाने पर निम्नलिखित आउटपुट प्राप्त होगा −

num =  36
Divisible by 3 and 2
....execution ends....

Else के साथ Nested if स्टेटमेंट

जैसा कि पहले उल्लेख किया गया है, हम if स्टेटमेंट के अंदर if-else स्टेटमेंट को nest कर सकते हैं। यदि if condition true है, तो पहला if-else statement execute होगा, अन्यथा else block में दिए गए स्टेटमेंट्स execute होंगे।

Syntax

if expression1:
    statement(s)
    if expression2:
        statement(s)
    else:
        statement(s)
else:
    if expression3:
        statement(s)
    else:
        statement(s)

उदाहरण

अब एक Python कोड लेते हैं जिससे यह समझा जा सके −

num = 8
print("num =", num)
if num % 2 == 0:
    if num % 3 == 0:
        print("Divisible by 3 and 2")
    else:
        print("divisible by 2 not divisible by 3")
else:
    if num % 3 == 0:
        print("divisible by 3 not divisible by 2")
    else:
        print("not Divisible by 2 not divisible by 3")

जब उपरोक्त कोड को निष्पादित किया जाता है, तो यह निम्नलिखित आउटपुट देता है −

num = 8
divisible by 2 not divisible by 3
num = 15
divisible by 3 not divisible by 2
num = 12
Divisible by 3 and 2
num = 5
not Divisible by 2 not divisible by 3
