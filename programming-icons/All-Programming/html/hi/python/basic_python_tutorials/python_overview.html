Python का परिचय

Python एक उच्च-स्तरीय, इंटरप्रेटेड, इंटरैक्टिव और ऑब्जेक्ट-ओरिएंटेड स्क्रिप्टिंग भाषा है। Python को अत्यधिक पढ़ने योग्य बनाया गया है। यह अक्सर अंग्रेजी कुंजीशब्दों का उपयोग करता है जबकि अन्य भाषाएँ विराम चिह्नों का उपयोग करती हैं, और इसमें अन्य भाषाओं की तुलना में कम सिंटैक्स संरचनाएँ हैं।

  • Python इंटरप्रेटेड है – यह रनटाइम पर इंटरप्रेटर द्वारा प्रोसेस होती है।
  • Python इंटरैक्टिव है – आप सीधे इंटरप्रेटर से संवाद कर सकते हैं।
  • Python ऑब्जेक्ट-ओरिएंटेड है – यह ऑब्जेक्ट-ओरिएंटेड प्रोग्रामिंग को सपोर्ट करता है।
  • Python शुरुआती लोगों के लिए उपयुक्त है – शुरुआती प्रोग्रामर्स के लिए बेहतरीन।

Python एक ओपन-सोर्स और क्रॉस-प्लेटफॉर्म प्रोग्रामिंग भाषा है। यह Python Software Foundation License (जो GNU General Public License के अनुकूल है) के अंतर्गत उपलब्ध है, और यह Linux, Windows और Mac OS जैसे प्रमुख ऑपरेटिंग सिस्टम्स पर उपलब्ध है।

नई सुविधाओं को आसान बनाने और पठनीयता को बनाए रखने के लिए Python Enhancement Proposal (PEP) प्रक्रिया विकसित की गई थी। इस प्रक्रिया के तहत कोई भी नई सुविधा, लाइब्रेरी या अन्य जोड़ के लिए PEP सबमिट कर सकता है।

Python की डिज़ाइन फिलॉसफी सादगी, पठनीयता और स्पष्टता पर बल देती है। Python को इसके "बैटरियाँ शामिल हैं" दृष्टिकोण के लिए जाना जाता है, क्योंकि Python सॉफ़्टवेयर के साथ एक व्यापक मानक पुस्तकालय वितरित किया जाता है।

Python की डिज़ाइन फिलॉसफी Zen of Python में दर्ज है। इसमें निम्नलिखित उन्नीस सूक्तियाँ होती हैं –

  • सुंदर बदसूरत से बेहतर है
  • स्पष्ट अप्रत्यक्ष से बेहतर है
  • सरल जटिल से बेहतर है
  • जटिल जटिलतापूर्ण से बेहतर है

Zen of Python दस्तावेज़ को पूरी तरह प्राप्त करने के लिए, Python Shell में import this टाइप करें –

>>>import this
